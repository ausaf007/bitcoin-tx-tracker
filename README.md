## How To Use In Dev?

1. Clone the repo and navigate to `bitcoin-tx-tracker/`:
   ``` 
   cd /path/to/folder/bitcoin-tx-tracker/
   ```
2. Get dependencies:
   ``` 
   npm install
   ```
3. Fill in the env file with the .env.sample provided, with the firestore credentials
4. Run the app:
   ``` 
   npm run dev
   ```
